<h1> Noobas Studio </h1>
Noobas Studio - software designed to help everyone to make manual translations, in first YouTube videos from ENG to RU.<br/>
<h3> Stack: </h3> C# (.NET Framework 4.8) - WPF, MaterialDesign, simple MVVM-architecture.
